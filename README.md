# OCR Metadata Audit Pipeline 🚀

**End-to-end OCR + metadata extraction for theatre posters**

## Features
- Tesseract OCR with preprocessing
- EXIF metadata extraction
- Structured field parsing (names, dates, orgs)
- Layout analysis (9-region grid)
- Quality scoring & uncertainty reports
- JSON/CSV/Markdown/HTML outputs
